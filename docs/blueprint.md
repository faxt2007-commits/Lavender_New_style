# **App Name**: SparkMatch

## Core Features:

- User Profile Creation: Allow users to create detailed profiles with photos, bios, and interests.
- Matching Algorithm: Suggest potential matches based on compatibility criteria (location, interests, preferences).
- Real-time Chat: Enable users to chat with their matches in real-time.
- Location Services: Use the geolocation feature to connect users in the vicinity.
- Interest-based Groups: Let users join groups based on interests and find people who share their passions.
- AI-Powered Icebreaker Generator: Suggest icebreaker messages for matches using a generative AI tool, making it easier to start conversations.
- Profile Recommendations: Dynamically suggest tips and refinements for user profiles using generative AI; for instance suggesting they add specific hobbies, experiences, or photos to increase their chances of finding compatible partners. Act as a tool to analyze a profile and suggest possible content improvements.

## Style Guidelines:

- Primary color: Coral pink (#F08080) to evoke warmth and romance.
- Background color: Light pink (#FAEBD7) to create a soft, inviting atmosphere.
- Accent color: Peach (#FFDAB9) to highlight key features and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif, for legibility.
- Use simple, modern icons related to love, connection, and shared interests.
- Prioritize a clean, card-based layout to display user profiles and other content.
- Implement subtle animations and transitions for a fluid, engaging user experience.
